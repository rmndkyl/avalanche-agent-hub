Avalanche Agent Hub Docs

Open-source developer toolkit for deploying AI agents and managing multi-wallet operations on Avalanche.

Installation

npm install @agent-hub/core

Quick Start

import { AgentHub } from '@agent-hub/core';

const hub = AgentHub({
  network: 'mainnet',
  rpcUrl: 'https://api.avax.network/ext/bc/C/rpc'
});

// Create a quest
const quest = await hub.quest.create({
  title: "Complete 5 swaps on Trader Joe",
  chain: "C-Chain",
  tasks: [
    { type: "on-chain", contract: "0x...", method: "swap", count: 5 },
    { type: "social", platform: "twitter", action: "follow" }
  ],
  rewards: { token: "AVAX", amount: 0.1 }
});

// Verify completion
const status = await hub.quest.verify({
  questId: quest.id,
  wallet: "0x..."
});

Modules

Quest SDK

Create, distribute, and verify on-chain quests.

// Create a quest
const quest = await hub.quest.create({ ... });

// Verify completion
const status = await hub.quest.verify({ questId, wallet });

// Get quest details
const details = await hub.quest.getQuest(questId);

// List quests
const quests = await hub.quest.listQuests({ status: 'active' });

Agent Kit

Deploy and manage AI agents on Avalanche.

// Deploy an agent
const agent = await hub.agent.deploy({
  type: 'trader',
  wallet: encryptedWallet,
  rules: {
    maxTradeSize: '100 USDC',
    allowedTokens: ['AVAX', 'USDC'],
    stopLoss: '5%'
  }
});

// Check status
const status = await hub.agent.status(agent.id);

// Pause/Stop
await hub.agent.pause(agent.id);
await hub.agent.stop(agent.id);

Wallet Manager

Multi-wallet management with encryption.

// Import wallets
const wallets = await hub.wallet.import({
  file: './accounts.json',
  encryption: 'AES-256',
  password: 'your-password'
});

// Get portfolio
const portfolio = await hub.wallet.portfolio(wallets);

// Get balance
const balance = await hub.wallet.getBalance('0x...');
